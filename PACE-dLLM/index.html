Diffusion Language Models

PACE-dLLM: Elastic Block Decoding via Confidence Cliff Estimation

Xiaocheng Lu

An inference-time decoder that reads the next reliable look-ahead horizon from a diffusion language model's own confidence curve.

ResultsBibTeX

Abstract

Diffusion language models support native parallel decoding, but common block-wise decoders use one block size to perform two roles: deciding how far to look ahead and deciding which tokens to commit. PACE-dLLM fits a context-dependent confidence cliff at each step, sets the next horizon by its saturation point, and uses a separate confidence threshold for commitment.
