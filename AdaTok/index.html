Adaptive Visual Tokenization

AdaTok: Self-Budgeting Image Tokenization with Quality-Preserving Dynamic Tokens

Xiaocheng Lu, Shuyang Guo, Xinyu Yue, Tao Han, Zhiwei Wang, Li Bai

A discrete 1D tokenizer that predicts a content-adaptive token budget in one pass.

ResultsBibTeX

Abstract

Image tokenizers typically encode every image with the same fixed number of tokens. AdaTok asks whether a discrete visual tokenizer can budget itself in one pass. It combines prioritized representation learning, which keeps prefixes decodable across budgets, with adaptive token allocation, which learns which prefix each image needs.
